Tagging is in full swing.

- [del.icio.us](http://del.icio.us/), [Furl](http://www.furl.com/), [My Web 2.0](http://myweb2.search.yahoo.com/), [Spurl](http://www.spurl.net/) etc. tags URLs
- [Yummy](http://yummy.printfu.org/) tags PDFs
- [Rojo](http://www.rojo.com/) tags RSS feeds
- [Tagcloud](http://tagcloud.com/), [24 eyes](http://www.24eyes.com/), [Feedmarker](http://www.feedmarker.com/), etc. tag RSS posts
- [Technorati](http://www.technorati.com/) tags blog posts
- [Tagsurf](http://tagsurf.com/) tags discussions
- [Job Bazaar](http://www.jobazaar.com/) tags job postings
- [Gmail](http://gmail.google.com/) tags e-mails (but you can't share tags)
- [Connotea](http://www.connotea.org/) and [CiteULike](http://www.citeulike.org/) tag academic references
- [Swik](http://www.swik.net/) tags open-source projects
- [Snippets](http://www.bigbold.com/snippets/) tags source code
- [43 things](http://www.43things.com/) tags things you want to do
- [43 places](http://www.43places.com/) tags places you want to visit
- [Diggs](http://digg.com/) tags stories
- [Library Thing](http://www.librarything.com/) tags books
- [Reader 2](http://reader2.com/) tags books
- [Upcoming](http://upcoming.org/) tags events
- [Dinnerbuzz](http://www.dinnerbuzz.com/) tags restaurants
- [Flickr](http://www.flickr.com/) tags photos
- [Tagzania](http://www.tagzania.com/) tags locations
- [Freesound](http://freesound.iua.upf.edu/) tags sounds
- [Podcast](http://www.podcast.com/) tags podcasts
- [Upto 11](http://www.upto11.net/) tags P2P music
- [Music mobs](http://www.musicmobs.com/) and [Genie Lab](http://genielab.com/) tag music
- [You tube](http://www.youtube.com/) tags videos

In fact, [supr.c.ilio.us](http://supr.c.ilio.us/about) tags tagging sites!
